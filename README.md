# HTML-Concepts
HTML Roadmap for Fresher – Topics to Cover from Basic to Advanced

1. Introduction to Web & HTML
	•	What is the Internet & Web?
	•	Client–Server architecture
	•	What is HTML?
	•	How a website works (Browser, URL, Hosting)
	•	What is HTTPS / HTTP?



2. Basic HTML Structure
	•	<!DOCTYPE html>
	•	<html>, <head>, <body>
	•	Meta tags – <meta> (charset, viewport, author, keywords, description)
	•	How to open and view a webpage in a browser



3. Fundamental HTML Tags
	•	Headings: <h1> – <h6>
	•	Paragraph: <p>
	•	Line breaks & spacing: <br>, <hr>
	•	Text formatting: <strong>, <b>, <i>, <em>, <u>, <mark>, <small>
	•	Comments: <!-- comment -->



4. Links & Navigation
	•	<a> tag – internal & external links
	•	Open link in new tab: target="_blank"
	•	Email/phone links



5. Images & Multimedia
	•	<img> tag + attributes (src, alt, width, height)
	•	<video>, <audio>, <source>



6. Lists
	•	Ordered list: <ol>
	•	Unordered list: <ul>
	•	Description list: <dl>



7. Tables
	•	<table>, <tr>, <td>, <th>
	•	Row/column span
	•	Table styling basics



8. Forms & Input Controls
	•	<form>, <input>
	•	Attributes: (name, value, placeholder, required, readonly)
	•	Form elements: <textarea>, <select>, <option>, <button>, <label>
	•	Types: text, password, email, checkbox, radio, file, date, range, etc.



9. Semantic HTML (Very Important for Interview & SEO)
	•	<header>, <nav>, <section>, <article>, <aside>, <footer>
	•	<main>, <figure>, <figcaption>
	•	Importance for SEO & accessibility



10. HTML Layout & Structure
	•	Page layout structure
	•	Container concepts with <div> and semantic tags



11. HTML Entities
	•	&nbsp;, &copy;, &reg;, &lt;, &gt;, etc.



12. Accessibility (A11Y) Basics
	•	alt attribute
	•	title attribute
	•	ARIA roles basics



13. Best Practices
	•	Clean and readable code
	•	Proper nesting & indentation
	•	SEO-friendly structure
	•	Responsive page structure



14. Practical Projects to Build
